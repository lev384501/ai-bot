# Minecraft AI Bot для Aternos

AI помощник для Minecraft сервера на Aternos.

## Сервер
- **IP:** aiserver1245.aternos.me
- **Версия:** 1.12.2
- **Тип:** Ванила

## Команды в игре:
- `бот помощь` - показать команды
- `бот иди` - идти вперед
- `бот стой` - остановиться
- `бот прыгай` - прыгать
- `бот найди дерево` - найти деревья
- `бот говори [текст]` - сказать текст

## Запуск:
1. Откройте репозиторий в GitHub Codespaces
2. Запустите: `python bot.py`
3. Включите сервер на Aternos
4. Бот автоматически подключится!
